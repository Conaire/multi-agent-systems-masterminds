Mastermind

Mastermind is a code breaking game between two players, based on the pencil and paper game Cows and Bulls. The colored peg and board version of the game was invented by an Israeli postmaster and telecommunications expert Mordechai Meirovits in 1970. The main theme of the game is that one of the players generate a code (playing the role of a Code Maker) and hides it from the other player. The opponent (playing as the Code Breaker) will then have to find the code based on guesswork and the feedback thus generated. The game consists of the following components.

  • Game Board
  • Code pegs (of upto 6 different colors)
  • Key pegs (of the colors Black and White)
The feedback generated provides the following information:
  • Black pegs are returned when the position and color of a code peg match the code set by the code maker
  • White pegs are returned when the color of a code peg is present in the code set by the code maker
  • At any given point in the game, a maximum of 4 key pegs (black, white or a mixture thereof) will be returned as feedback for the guess in that round

Mastersminds

Mastersminds[1] is an extension of Mastermind, in which players take turns in guessing the opponent's code. Thus the game includes the following changes:

  1. Each player sets a separate code simultaneously at the beginning of the game.
  2. Each round of the game is composed of two sets of guesses, with each player guessing the opponent's code.
  3. At the end of each guess, there are two sets of feedback.
    • Opponent provides feedback on player's guess by comparing it to opponent's original code.
    • Player provides feedback on player's guess comparing the guess to player's own original code.
    This creates an exchange of knowledge between players while players also try to minimize the knowledge they give away to the opponent (in terms of feedback on their own guess.)[2]
  4. The player that guesses the opponent's code first wins the game.
